# My_hunter

My_hunter is a shooter based on the duck hunt game

## Installation

Use command Make for compilation a binairy

```bash
make
```

## Usage

For help you can use:

```bash
./my_hunter -h
```

For playing you can use:

```bash
./my_hunter
```
